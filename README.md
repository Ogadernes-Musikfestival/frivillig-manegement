This is a small simple Internal admin system for managing volunteers (Frivillige) at a music festival.
The system is intentionally simple, low-cost, and designed for a small group of management users.
Build using next.js.

Purpose

- Collect volunteer signups from the public festival website
- Give management an internal tool to:
- View volunteers
- Read preferences and comments
- Manually create/edit volunteers
- Assign and organize volunteers during the festival
